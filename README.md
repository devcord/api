# Devcord API
## prerequisites
- node.js 12+

## tech stack
- typescript
- koa
- discord api

## installation
```
git clone https://github.com/devcord/api devcord-api
cd devcord-api
npm install
```

## commands
```js
{
  "start" // starts the server
  "dev" // starts the server in development mode
  "build" // builds the server
  "lint" // lint and fix
}
